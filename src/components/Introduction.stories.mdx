# Installation

>  Support `Ver. React > 17` 

```sh
yarn add nexpie-component-library
```


## Usage

```jsx
import { IndicatorLight } from "nexpie-component-library"";

ReactDOM.render(<IndicatorLight />, mountNode);
```


**Source Code** [**gitlab**](https://dev.nexpie.com/phisek/nexpie-component-library)
